# 减少bug
>方法：拿出一张纸，写出每层的各个矩阵的维度（shape）

x = (n^[0], 1)     
w^[1] = (n^[1], n^[0])  
z^[1] = (n^[1], 1) 
b^[1] = (n^[1], 1) 
> z^[1] = w^[1].T * x  + b  
> 即 (n^[1], n^[0])   *   (n^[0], 1) 


一般的：   
a = (n^[L-1], 1)     
w^[L] = (n^[L], n^[L-1])  
z^[L] = (n^[L], 1) 
b^[L] = (n^[L], 1) 
> z^[L] = w^[L].T * a 
> 即 (n^[L], n^[L-1])   *   (n^[L-1], 1) 

向量化(m 为训练集样本个数)  

A = (n^[L-1], m)     
W^[L] = (n^[L], n^[L-1])  
Z^[L] = (n^[L], m) 
b^[L] = (n^[L], 1) 
> Z^[L] = W^[L].T * A  
> 即 (n^[L], n^[L-1])   *   (n^[L-1], m)    


dw 维度 = w   
db 维度 = b

# 构建神经网络的一般步骤
- 定义神经网络结构（如输入层、输出层、隐藏层的参数个数）
- 初始化参数（权重W要随机初始化、b可初始化为0）
- 循环：前项传播，计算损失；后向传播，应用梯度下降，更新参数（W、b）
