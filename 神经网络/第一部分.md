# 减少bug
>方法：拿出一张纸，写出每层的各个矩阵的维度（shape）

X = (n^[0], 1)     
W^[1] = (n^[1], n^[0])  
Z^[1] = (n^[1], 1) 
b^[1] = (n^[1], 1) 
> Z^[1] = W^[1].T * X  + b
> 即 (n^[1], n^[0])   *   (n^[0], 1) 

一般的：   
A = (n^[L-1], 1)     
W^[L] = (n^[L], n^[L-1])  
Z^[L] = (n^[L], 1) 
b^[L] = (n^[L], 1) 
> Z^[L] = W^[L].T * X  
> 即 (n^[L], n^[L-1])   *   (n^[L-1], 1) 

dW 维度 = w
dB 维度 = b
